
🌍 **Global Government Budget Analytics Dashboard**

📌 Project Overview

The Global Government Budget Analytics Dashboard is an interactive Power BI dashboard designed to analyze government budget allocation across multiple countries from 2021 to 2026.

The dashboard helps users understand how different governments distribute their budgets among major sectors such as:.
🛡 Defense.
🎓 Education.
🏥 Health.
🏗 Infrastructure.
🤝 Social Welfare.
🌾 Agriculture.

Using interactive filters, maps, KPIs, and visualizations, decision-makers can compare countries, analyze spending trends, and identify budget priorities

🎯 **Business Problem**

  Governments allocate billions of dollars every year across different sectors. However, comparing spending patterns across countries is difficult due to scattered data.

**This dashboard solves that problem by providing:**

-> Country-wise budget comparison
-> Year-wise spending analysis
-> Budget allocation by sector
-> Top spending countries
-> Budget growth trends
-> Interactive global map

📊 **Dashboard Features**

🌍 **Country Filter**
Allows users to analyze individual countries.

📅 **Year Filter**
Compare budgets between different years (2021–2026).

💰 **KPI Cards**
-> Total Budget
-> Average Budget
-> Total Countries

🗺 **Interactive Map**

Displays countries geographically for better visualization.


📈 **Spending Year Wise**

Shows annual spending across
--> Defense

📋 **Country Budget Summary**

Displays detailed budget values for every country
